# Proposed pilot bill of materials

The list below is a planning structure. Supplier quotations and exact models must be verified before procurement.

| Item | Purpose | Pilot quantity |
|---|---|---:|
| Raspberry Pi or industrial edge computer | Local acquisition, validation and buffering | 1 |
| High-endurance storage | Operating system, logs and temporary buffer | 1 |
| Approved DC power supply | Stable low-voltage gateway supply | 1 |
| Isolated USB-to-RS485 adaptor, where required | Modbus RTU communications isolation | 1 |
| Ethernet cable or approved network adaptor | Connection to the institutional network | 1 |
| Enclosure and DIN-rail mounting accessories | Physical protection and installation | 1 set |
| Optional energy submeter and CTs | Additional visibility where existing meters are insufficient | As approved |
| Protection and termination accessories | Safe low-voltage installation | As designed |

No contactor, relay or mains switching component is required for the current monitoring-only demonstration.
