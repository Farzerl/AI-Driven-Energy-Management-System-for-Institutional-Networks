# Live Inference Validation

This folder contains public-safe validation results for the operational edge-to-AI path.

- `public_demo_model_metrics.json`: synthetic fallback model test results.
- `private_training_pipeline_test.json`: aggregate confirmation that the private UZ ZIP parser, chronological split, training and test workflow completed successfully. No raw readings, real facility map or private model is included.
- `end_to_end_live_inference_test.json`: API and simulated gateway result generated during package validation.
